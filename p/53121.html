领克安装百度地图V20大屏车机板，老车机用户有福了~32位百度大屏版它来了

2024-07-26T13:52:51.000Z2024-07-26T14:05:45.564Z8252 min深圳

星期五，百度大屏版又又又更新了！每周百度在群里都会听取用户建议，这不，V20.1.4 它来了。

本次更新的重点

百度地图 V20 车机 32 位版终于来啦！
支持更多可以安装 app 的老车机用户。

其它更新

  1. 优化 UI 错位问题
    针对上个版本开放系统状态栏后，部分车机系统画中画启动导航后，界面 UI 错位的问题进行了优化。工程师研究了各种系统的特性，最新版无论是否开启系统状态栏，都会正常显示【手车互联】、【GPS 信号状态】等。大家可以开启车机端 APP，在【设置】-【显示】-【页面最下方】-【应用全屏显示开关】自行设定。

  2. 比例尺及地图显示优化
    针对巡航的比例尺、地图显示范围、俯视角、透视角，进行了“小数点级”的微调，使局部内容更清晰。

  3. 解决闪退问题
    解决了部分车机在导航中、前后台切换时偶现的闪退问题，提升了稳定性。遇到类似问题的小伙伴请多关注此更新。

  4. 修复其他问题
    解决了一些隐藏问题，进一步提高整体使用体验。

百度大屏版功能亮点

1. 车道级导航
支持 3D 车道级导航，清晰地显示路况、路口、转弯等信息。遇到事故、施工等突发性道路交通事件时，为用户提供车道级预警服务。

2. 红绿灯倒计时
拥有超过百万的红绿灯大数据，结合城市交通运行数据进行分析理解；实时生成红绿灯倒计时读秒和绿波车速建议，方便车主控制安全出行。

3. 多明星语音包
提供海量热门影视、明星、二次元等多种风格语音包，大屏版同步拥有，随心更换。

4. 手车互联
实时联动导航：手机上搜目的地、路线规划等信息能自动同步到车机大屏上；数据双端同步：手机版和大屏版的收藏夹、常用地址、足迹等信息自动同步。

5. 智能语音助手
支持“小度小度”全端语音唤醒，可以发起导航、多途径点路线查询，以及地点、限号、天气等语音查询；导航中同步支持目的地变更、路线切换、路况查询等操作；动口不动手，驾驶更安全、操作更便捷。

6. 自动巡航功能
在大屏版上车后会自动进入巡航模式；巡航状态下也支持红绿灯雷达功能，巡航自动实时探测前方红绿灯情况，并显示读秒。

领克车机声道选择

针对领克车机的声道问题，我们可以在设置里面连续点击播报来选择 11 声道或者 2 声道。
