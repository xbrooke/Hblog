网易Filmly已上线WebDAV功能，支持iOS/安卓/TV端

2024-07-29T14:45:38.000Z2024-07-30T14:15:53.646Z1.3k4 min深圳

作为一个资深影迷，我不知道大家平时有什么休闲娱乐的方式？你是喜欢宅在家里吹着空调，喝着快乐水，一边刷短视频玩游戏，还是喜欢和亲友外出旅游，领略大好河山呢？

对于我来说，看剧是首选。理由很简单，看剧不受时间和地点的限制，而且几乎没有成本。

看剧设备也很灵活，投影仪、电脑、平板、手机都可以成为观影神器。

对于许多使用 NAS 的朋友而言，影音需求通常是首要考虑的。为了获得良好的影音体验，选择一款优秀的影音播放工具是基本操作。

最近，阿里云开始对“第三方应用权益”进行收费，这对许多用户来说是一个挑战。而此时，网易推出了其媒体库播放器应用“网易 Filmly”，并上线了 WebDAV 功能，为用户提供了更多接入方式。

网易 Filmly V1.3.0 更新内容

  1. 新增令人期待的 WebDAV 功能，连接 WebDAV 后可随时随地畅享海量高品质影片。
  2. 优化网盘刮削机制，提升刮削稳定性。
  3. 优化首页刮削提示，资源数量和任务状态展示更清晰。
  4. 修复外挂字幕失效问题。
  5. 修复部分兼容性问题。

接下来，我们来看看如何在 Windows 上部署 Alist 网盘。

下载 Alist

在 GitHub 上下载 Alist 压缩包，根据自己的系统选择 32 位或 64 位版本进行下载。 点击下载

软件包名称 对应系统版本
alist-windows-386.zip Windows 32 位系统
alist-windows-amd64.zip Windows 64 位系统
alist-windows-amd64-upx.zip Windows 64 位系统(upx 压缩)
alist-windows-arm64.zip Windows arm64 架构系统

点击 Releases，进入下载页面：

向下滚动，找到 Windows 的 zip 包，根据自己的系统版本选择对应的版本（此处以 alist-windows-amd64.zip 为例）。

如何查看自己系统的位数？按下 Win(徽标键)+R，输入 cmd，然后在命令行窗口中输入：wmic os get osarchitecture

根据你的储存需求，在磁盘中创建一个名为 alist 的文件夹。解压下载的压缩包，并将其内容解压到该文件夹中。你会看到一个名为 alist.exe 的文件，如下图所示：

在文件夹地址栏中输入 cmd，然后按 Enter 键打开命令行窗口，以确保命令行的当前目录是 Alist 程序所在目录。

挂载网盘

目前，Alist 支持众多主流网盘，如百度网盘、阿里云网盘、夸克网盘、115 网盘等。具体详情请查看 官方文档。

登录 Alist 管理后台，找到“储存”选项，添加对应的网盘（本文以夸克网盘为例）。

挂载路径填写为 /夸克网盘，序号为 1。

挂载夸克网盘需要在网页端登录夸克网盘，获取 Cookie。

在键盘上按 F12 进入开发者模式，找到 网络，刷新网页，找到 sort? 开头的文件，点击进去，找到 Cookie 并复制对应部分的内容。

将对应的 Cookie 粘贴到相应的区域，然后点击保存。

此时，夸克网盘已成功挂载到 Alist，其他网盘的挂载方法请查看 官方文档。

Filmly 连接 WebDAV

请将 Windows 防火墙关闭或将 Alist 添加到白名单（这一步尤其重要，否则无法在局域网内登录 Alist）。

打开 Filmly，添加新资源，选择“从 WebDAV 导入”。

地址填写为部署 Alist 的电脑局域网 IP，如（192.168.31.74），端口填写 5244，用户名和密码填写你刚才的 Alist 登录账号和密码，路径填写为 /dav，确认添加即可。

选择添加夸克网盘。如果添加了多个网盘，请根据需要选择对应的网盘。

此时，挂载的夸克网盘正在扫描并刮削，快去享受 Filmly 带来的快乐吧！
